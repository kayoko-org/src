openXào
======

openXào is a POSIX-compliant, fast, secure, and highly portable Unix-like operating system.

Binaries
--------
- [Releases](https://cdn.openxao.net/releases/latest)

Testing
-------

On a running openXào system:

    make -C /usr/tests [posix|sus|legal]
