Kash - The Korn Again Shell


Kash is a shell designed for UNIX systems. It is similar in vein to the Korn Shell (by David Korn).

Directory / File Type Responsibility
inc/sh/ Directory Header files (Class definitions and interfaces)
src/ Directory Implementation files and compiled object code
inc/sh/command.hh Header Base class for AST nodes (Command Pattern)
inc/sh/lexer.hh Header Tokenization logic and string expansion (PS1/PS2)
inc/sh/parser.hh Header Recursive descent or shift-reduce parser interface
src/main.cc Source REPL loop, signal handling, and POSIX built-ins
src/pipeline.cc Source Inter-process communication (pipes) and fork/exec
Makefile Build Compilation instructions for the Solaris environment
sh Binary The final executable ELF (linked C++ application)
