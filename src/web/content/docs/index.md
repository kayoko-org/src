<p class="lead text-xl text-slate-400">
  Kayoko is a POSIX-compliant, fast, secure, and highly portable Unix-like system.
</p>

## Features

* **RBAC** (via KAT - Kernel Access Table)
* **SMAT** (System Manager for Administrative Tools)
* NetBSD-based foundations
